Callisto Light & plentymarkets CMS Tools

Mit Callisto Light wurde das Standard Shop-Design vom plentymarkets Webshops entwickelt.
Das Projekt basiert auf dem internen Content-Management-System von plentymarkets. Für ein responsives Layout wurde die UI auf Basis des CSS-Frameworks Twitter Bootstrap 3, für dynamische Funktionalitäten und die asynchrone Kommunikation mit dem Webserver die plentymarkets CMS-Tools entwickelt. Diese JavaScript-Bibliothek basiert auf jQuery und implementiert u.a. einen eigenen Dependency-Injection-Mechanismus sowie Möglichkeiten zur direkten Verknüpfung von UI-Elementen mit bereitgestellten Funktionen direkt im Markup.
